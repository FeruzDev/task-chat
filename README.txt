
1. Сначала перейдите в раздел "client" и введите команду "npm install"
2. Затем перейдите в раздел «сервер» и введите «npm install»,a потом  введите команду "node index", чтобы запустить сервер.
3. Перейдите в папку  "client" и введите "npm start"
4. после запуска проекта, вам нужно ввести «User name» и «Room ID», чтобы получить доступ к чату.
5. Для того чтобы общаться с кем то отдельно нужно ввести одинаковую «Room ID» обоим собиседникам.
